Admin Template之main-page
----  
基于 vue, vue-router, [admin-toolkit](http://gitlab.skysri.com/liangweiliang/admin-toolkit),[admin-template](http://gitlab.skysri.com/liangweiliang/admin-template) 的管理后台代码模板主题一。  

## 使用  

```
git clone  
cd main-page
npm install  
```
```
// 开发  
npm run serve  
```

```
// 构建  
npm run build


```
